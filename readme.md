# Solar Photovoltaic Yield & Thermal Efficiency Calculator

A Python-based engineering project that estimates photovoltaic energy generation while accounting for temperature-induced efficiency losses.

## Features

- User input for panel specifications
- Hourly environmental data entry
- Solar irradiance analysis
- Cell temperature estimation
- Thermal derating calculations
- Power and energy generation calculations
- Data visualization using Matplotlib

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib

## Engineering Concepts

The project models photovoltaic performance using:

- Solar irradiance
- Nominal Operating Cell Temperature (NOCT)
- Temperature coefficient of power
- Thermal derating effects

## Generated Visualizations

1. Solar Irradiance vs Hour
2. Actual Power vs Hour
3. Base Power vs Actual Power
4. Efficiency vs Cell Temperature

## Example Usage

```bash
python solar_pv_calculator.py
```
